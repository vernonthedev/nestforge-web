# Welcome to NestForge Web

NestForge Web is a blazing-fast fullstack framework that combines NestForge (Rust) backend with Next.js-style frontend for rapid development.

## Getting Started

1. **[Quick Start Guide](quick-start)**: Get up and running in minutes
2. **[CLI Reference](cli)**: Full CLI command documentation  
3. **[Routing Guide](routing)**: Learn file-based routing

## Why NestForge Web?

- **Fast**: Rust-powered backend with minimal overhead
- **Productive**: Next.js-style file-based routing for frontend
- **Type-Safe**: Full Rust type safety end-to-end
- **Flexible**: Mix and match frontend/backend as needed

## Quick Start

```bash
# Install CLI
cargo install --path crates/nestforge-web-cli

# Create project
nestforge-web new my-app
cd my-app

# Start dev server
nestforge-web dev
```
